## 🚀 Como Começar Rápido

### 1. Abrir a Página

Abra `index.html` no navegador (Chrome, Firefox, Safari).

### 2. Testar Tudo

- Clique para abrir o envelope
- Role para ver todas as seções
- Clique no botão 🎵 para música
- Teste no celular também!

### 3. Adicionar Uma Música

**Aba de Música Única:**

1. Clique em 🎵 (canto inferior direito)
2. Certifique-se que está na aba **🎶 Música**
3. Cole uma URL do Spotify ou YouTube
4. Clique "Adicionar 🎶"
5. Clique "▶️ Tocar"

**Aba de Playlist/Álbum:**

1. Clique em 🎵
2. Clique na aba **📋 Playlist**
3. Cole uma URL de álbum ou playlist do Spotify
4. Clique "Carregar 📀"
5. O player aparece! Clique em qualquer faixa

## 🎨 Customizações Mais Comuns

### ✏️ Editar o Texto da Carta

Abra `index.html` → Procure por `.carta-section` → Edite o parágrafo

### 🖼️ Adicionar Uma Foto

Procure por `photo-placeholder` e substitua o emoji pela imagem

### 🌈 Mudar Cores

No CSS, encontre `:root` e altere os valores HEX

### 🎵 Adicionar Música Permanente

Coloque um MP3 na pasta e edite a tag `<audio>`

**Guia completo**: Leia [CUSTOMIZACAO.md](docs/CUSTOMIZACAO.md)

## 🎵 Música - Passo a Passo

### Do Spotify:

1. Abra Spotify
2. Escolha uma música
3. Clique "Compartilhar" → "Copiar link da música"
4. Cole em: botão 🎵 → input

### Do YouTube:

1. Find o vídeo no YouTube
2. Compartilhar → Copiar link
3. Cole no botão 🎵

**Mais info**: Veja [MUSICA.md](docs/MUSICA.md)

## 🛠️ Troubleshooting

| Problema          | Solução                          |
| ----------------- | -------------------------------- |
| Carta desalinhada | `line-height` em CSS `.carta`    |
| Mobile deformado  | Testar em 600px breakpoint       |
| Música não toca   | Verifique URL e se tem http://   |
| Fonte não carrega | Conexão internet vs Google Fonts |
| Animações lentas  | Desabilitar em `@media` mobile   |
