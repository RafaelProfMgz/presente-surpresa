# 🎀 Skill: Presente - Customização & Desenvolvimento

## 📋 O que faz

Esta skill é especializada em trabalhar com o projeto **Presente** - uma página web interativa criada para aniversário. Fornece:

- ✨ **Customizações Rápidas**: Alterar textos, cores, emojis
- 🎨 **Alterações de Design**: Fontes, tamanhos, espaçamento responsivo
- 🎵 **Integração de Música**: Spotify, YouTube, áudio local
- 📱 **Otimizações Mobile**: Garantir responsividade total
- 🚀 **Melhorias de Performance**: Animações, carregamento
- 🐛 **Troubleshooting**: Problemas comuns e soluções

## 🎯 Scope

### ✅ **Use esta skill quando:**

- Precisa editar texto da carta ou qualidades
- Quer mudar paleta de cores
- Adicionar/remover seções
- Otimizar para mobile
- Resolver bugs de alinhamento
- Integrar música externa
- Melhorar animações e UX
- Adicionar funcionalidades

### ❌ **Não use quando:**

- Criar um novo projeto zero
- Questões totalmente fora do escopo
- Backend/servidor (é HTML/CSS/JS puro)

## 📝 Tarefas Comuns

### 1. Editar Carta

**Localização**: Linha ~1194 do `index.html`  
**Ação**: Edite os `<p>` tags dentro de `.carta`

```html
<p>Novo texto aqui...</p>
```

### 2. Mudar Cores Globais

**Localização**: CSS `:root` (~linha 15)  
**Ação**: Altere valores HEX

```css
--rosa-forte: #f06292;  →  #your-color
```

### 3. Adicionar Emoji/Card

**Localização**: Seção relevante (qualidades, livros, vales)  
**Ação**: Duplicate e edite estrutura existente

### 4. Responsividade Mobile

**Localização**: `@media` queries (~linha 1040)  
**Ação**: Ajuste padding, font-size, layout para breakpoints

### 5. Sistema de Música

**JavaScript**: Funções `addMusicFromURL()`, `toggleAudio()`  
**Suporta**: Spotify e YouTube URLs

## 🚀 Padrão de Resposta

Ao trabalhar com requests desta skill:

1. **Entenda o escopo**: O que exatamente precisa ser mudado?
2. **Localize no código**: Encontre a seção relevante
3. **Implemente**: Faça changes diretas no arquivo
4. **Teste responsividade**: Verifique mobile (599px, 768px, 1024px)
5. **Confirme**: Mostre o resultado ou mudanças feitas

## 📱 Breakpoints

- **<600px**: Mobile pequeno (otimizar para thumbs)
- **600-768px**: Tablet pequeno
- **768-1024px**: Tablet grande
- **>1024px**: Desktop

## 🎵 Sistema de Música

### Formatos Aceitos:

- Spotify: `https://open.spotify.com/track/[ID]`
- YouTube: `https://youtu.be/[ID]`

### Implementar Nova Ação:

1. Extrair ID do URL
2. Validar formato
3. Atualizar `musicInfo` com feedback
4. Ativar botão reproduzir

## ⚠️ Considerações Importantes

- **HTML é único**: Tudo (HTML + CSS + JS) em 1 arquivo
- **Sem dependências**: Funciona 100% offline após load
- **Responsivo**: Testar SEMPRE em mobile
- **Performance**: Animações podem lag em celulares antigos
- **CORS**: YouTube Web Embed pode ter restricões

## 📞 Referência Rápida

| Precisa...       | Procure em...                        |
| ---------------- | ------------------------------------ |
| Alterar texto    | `.carta p`, `.recorte`, `.vale`      |
| Mudar cores      | `:root` do CSS                       |
| Adicionar emojis | Qualquer classe com `emoji`          |
| Mobile issues    | `@media (max-width: 600px)`          |
| Música           | Função `addMusicFromURL()`           |
| Animações        | `@keyframes` no CSS                  |
| Fontes           | `<link>` no head + CSS `font-family` |
